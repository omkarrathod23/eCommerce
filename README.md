Shofy is a modern full stack eCommerce web application with an admin panel, built using Next.js, Express.js, MongoDB, and Redux Toolkit.  
This project is set up locally for learning, customization, and further development.

---

## Tech Stack

### Frontend
- Next.js (React)
- TypeScript
- Redux Toolkit & RTK Query
- Bootstrap 5
- Sass (SCSS)

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Nodemailer
- Stripe Payment Gateway

---

## Core Features

- User authentication (Register / Login)
- Product listing & product details
- Shopping cart & checkout
- Stripe payment integration
- Admin panel for product & order management
- Responsive UI for all devices
- Secure REST APIs
